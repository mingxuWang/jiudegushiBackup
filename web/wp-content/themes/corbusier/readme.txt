== Description ==

Corbusier is a quality Wordpress Theme designed to be flexible for businesses. It is fully responsive and features a full screen background image slider.
Handcrafted and designed by Arcimedia, it has many customisable elements, including a dark and light style. Take a moment to explore the details.

== Installation ==

1. Upcompress `corbusier.zip` and upload it to the `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance - Themes' menu in WordPress
